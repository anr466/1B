# Master Patterns Skill

> Comprehensive patterns for thinking, planning, execution, and design.
> Extracted from Claude Code architecture + Trading Bot System + Frontend Design.
> Reusable across any project.

## Quick Start

Load this skill when working on:
- System architecture & initialization
- Multi-step planning & execution
- Frontend/UI design & implementation
- Backend/API development
- Security & testing

## Pattern Categories

See `master-patterns.md` for full details:

1. **Bootstrap & Initialization** — Ordered loading, deferred init, mode routing
2. **Execution & Query** — Turn-based execution, intent routing, backlog planning
3. **State & Session** — Session persistence, transcript compaction, permissions
4. **Frontend & UI** — State stability, RTL, responsive design, animations
5. **Backend & API** — REST design, error handling, database patterns
6. **Security** — Auth flow, API security, data protection
7. **Testing** — Coverage checks, re-audit requirements
8. **Workflow** — Intent classification, delegation, commits
9. **Quality Gates** — Zero tolerance, code quality checklist
10. **Quick Reference** — When to use each pattern
