# 📋 Spec Kit Skills for WindSurf

## 🎯 مهام Spec Kit المحسّنة لـ WindSurf

### 📋 spec-init: تهيئة المشروع
**متى تستخدم:** بداية أي مشروع جديد أو إعادة هيكلة
**ما يفعل:** يهيئ هيكل المشروع الأساسي

```
استخدم عندما تريد:
- بدء مشروع جديد
- إعادة هيكلة مشروع موجود
- إنشاء هيكل احترافي
```

### 📋 spec-create: كتابة المواصفات
**متى تستخدم:** بعد تهيئة المشروع
**ما يفعل:** يولد مواصفات تفصيلية للمشروع

```
استخدم عندما تريد:
- تحديد متطلبات المشروع
- كتابة مواصفات المستخدم
- تحديد معايير النجاح
```

### 📋 spec-plan: التخطيط التقني
**متى تستخدم:** بعد كتابة المواصفات
**ما يفعل:** يولد خطة تقنية شاملة

```
استخدم عندما تريد:
- اختيار التقنيات المناسبة
- تصميم البنية المعمارية
- تحديد قيود النظام
```

### 📋 spec-tasks: تقسيم المهام
**متى تستخدم:** بعد التخطيط التقني
**ما يفعل:** يقسم المشروع إلى مهام صغيرة

```
استخدم عندما تريد:
- تقسيم مشروع كبير
- تنظيم العمل الفريق
- تخطيط التنفيذ
```

### 📋 spec-review: مراجعة المواصفات
**متى تستخدم:** أثناء التطوير
**ما يفعل:** يراجع ويحسن المواصفات

```
استخدم عندما تريد:
- تحديث المواصفات
- مراجعة التقدم
- إضافة متطلبات جديدة
```
